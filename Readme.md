## 一代镖主控代码
- 项目来源于25-missle-V1,在此基础上进行修改。
## 代码版本
1. 标签：“可以连续发射”
- 进行了自检,校准,发射流程完整测试。
    自检直接跳过后的流程都没问题，校准完后会一直重复发射流程。
    只是此时的自检和校准都存在隐患，直到下一次标签才修复。
2. 标签：“校准，发射，调参板已验证”
- 进行了自检,校准,发射流程和摇杆/调参板控制落点的完整测试。
- 修复了失联后重新校准会导致电机撞到限位开关后不停的问题。
	原因是调用的start_calibration()没有重置校准状态标准位，导致给了速度却跳过了限位检查（内部会停下电机并切换电机模式）。
- 适配了调参板代码，优化了S2控制yaw状态的逻辑。
	现在校准后，三个拨杆位置直接对应三个状态。
	分别测试了对yaw角度和行程电机进行摇杆增量控制，调参板控制。并且都受到yaw子状态机内部设置的target限幅保护。
	视觉只看了接收包状态，end会一直变，其他都是0，不知道对不对。
- 测试了堵转保护逻辑和意外碰触限位开关的保护逻辑，后期考虑到实际工况后删除掉了。
- 增加了debug状态，可以在此状态下调试各电机pid。
	速度环无保护，需要额外注意。角度环有目标限幅，但是震荡起来也可能失去限幅。
- 自检逻辑有问题，自检完后会直接跳转校准，会导致最后一个按下的开关被意外识别。
	后面加了自检状态结束需要松开所有按键。还没测试
- 发射流程有问题，和机械对接时对装填理解有误，发现4发镖的流程没办法简单用一套发射状态机管理，后面为4发镖都设置了完整的状态机，等待测试。
- 滑块和行程丝杆在主状态机的角度环target没有限幅，存在超界风险。（不过在主状态机填位置都很少，而且都是用的宏参数）
	后面为了安全起见在发射类的adjust里面（最终计算pid的源头）对pid_angle的target进行了限幅。没有进行测试。
- 标志位存在部分没有显示初始化。后期全部进行了显示置false。没有测试。
- dr16类全局共用，没有互斥锁保护。后面增加了dr16快照copy，并且增加了互斥锁，没有测试。
- 后面由于4发发射流程不同，所以设置了连续开火限制，每4发就需要手动退出自动发射模式，然后重新进入自动发射4发流程。没有测试
	感觉这里会出新bug。
## 遥控使用：
关控电机全部失能，并且重新连接后会重新开始校准流程，校准后会继续发射流程。
    我认为上电后就不需要进行二次校准了，但是说是有可能会出现不同步的情况。
    所以还是保留了这样一种除了reset之外的校准触发方式。

| 摇杆  | 状态         | 动作                                                                 |     | 摇杆  | 状态     | 动作                             |
| --- | ---------- | ------------------------------------------------------------------ | --- | --- | ------ | ------------------------------ |
| s1  | up         | 电机失能（不修改状态）                                                        |     | s2  | up     | 手动控制yaw和igniter                |
|     | middle     | 进自检，自检手动完成或者跳过自动进校准，校准完后进等待发射，（卡stay状态，后续的autofire状态会因此被重置到stay状态） |     |     | middle | 使用修正值控制yaw和igniter（修正值由调参板设置）  |
|     | down       | 激活自动发射，（进autofireprep状态），跳过自检（自检就是手动按限位开关）                         |     |     | down   | 使用视觉控制，如果视觉失联，转而用修正值控制，直到视觉重连。 |
| LX  | left/right | 增量控制yaw                                                            |     | RX  |        |                                |
| LY  | up/donw    | 增量控制igniter                                                        |     | RY  |        |                                |
